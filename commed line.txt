//Git Hub
Git status = untuk memerika status file
Git add <namafile> = untuk menambahkan spesifik file
Git add . = untuk menambahkan semua file
        Ex: Git add main
Git commit –m “pesan commit” = untuk melakukan commit ke repository. 
       Ex: git commit –m “ini commit pertamaku”
Git push = untuk melakukan push ke repository

Git log --oneline = cek last commit (dengan one line)
git revert <hash commit> = membatalkan commit tertentu dengan membuat commit baru yang berlawanan
git remote -v = 
git checkout <no commit> = untuk masuk ke dalam commit
git rebase <main> =  untuk menggabungkan commit dari satu branch ke branch lain (timpa)

GIT branch =  untuk melihat semua branch
git checkout -b <namabranch baru> = untuk crete branch baru
git checkout <nama branch> = untuk pindah branch
Git cherry-pick <no commit> = untuk memilih satu atau lebih commit dari suatu branch dan menerapkannya ke branch lain
        Ex: Git cherry-pick 8541cd0
Git merge <namabranch> = menggabungkan branch (biasa di gunakan untuk merge dari branch biasa ke main)
        Ex: Git merge dev
